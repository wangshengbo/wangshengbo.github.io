根据完全二叉树的节点数计算二叉树的深度

标签（空格分隔）： 未分类

思路

具有N个节点的完全二叉树的深度为$\lfloor\log_2 N\rfloor+1$。

JAVA代码

import java.util.Scanner;
public class DeepOfTree {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int nodeNum;
        System.out.println("请输入完全二叉树的节点数：");
        nodeNum = scanner.nextInt();
        int deep = (int) (Math.log(nodeNum) / Math.log(2)) + 1;
        System.out.println("完全二叉树的深度为："+deep);
    }
}

牛客网编程题注意事项

Q:为什么代码在我本机可以通过，提交到牛客网总是提示答案错误

A:牛客网的每个编程题都有10-50组测试数据，你的代码在本机可以通过仅仅是通过了样例数据，可能存在边界条件考虑不周导致更多的测试数据没有通过，所以会返回答案错误。还有一种可能是你的代码没有处理多个测试用例，例如A+B的代码需要while循环处理
